我的博客是怎么搭建的？

Posted on 2024-04-18 In blog Views: Word count in article: 840 Reading time ≈ 3 mins.

博客， 是我的知识循环系统中的最后一环，也是最重要的一环，它承担我的输出，以及个人品牌。本篇文章，我介绍一下我的博客，是怎么搭建的。

概况

我的博客的架构是用 hexo+github+Cloudflare pages。 换句话说，除了域名需要收费外，其他的都是免费的。

博客架设

网络上关于使用这套架构搭建博客的文章很多，我就不再赘述了， 这里放几个我参考过的链接。

  • cloudflare: Deploy a Hexo site

主题

我是用的主题是社区的next-theme ，这个主题是在 theme-next （已经停止维护）上继续开发的。它的简约风格 以及文档 都是我喜欢的，所以我就使用了它。后面的一些配置也是根据next-theme 来的。

word count

使用hexo-word-counter统计文章的字数以及预期阅读时间。完成配置后，可以在每篇文章开头和页面底部显示字数和阅读时间

这样字数统计功能就完成了。

内容管理

我现在使用Obsidian 管理我的所有数字笔记，我的也是如此。
我利用Obsidian quickAdd + template 插件，一键就能创建一篇新博客。
图片等媒体文件我是托管再 Cloudflare R2。
使用了Obsidian 的 Image auto upload Plugin 插件 + picList 实现图片上传。 获得的写作体验 一点也不逊色与medium。

总结

到此，我的博客系统已经介绍完毕了，如果您对拥有一个自己的博客感兴趣，不妨动手试试。 这一切都是免费的。
